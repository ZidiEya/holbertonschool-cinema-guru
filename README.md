# 🎬 holbertonschool-cinema-guru

A React-based movie application built with Yarn, Axios, FontAwesome, and more. This project is part of Holberton School's web curriculum.

---

## 🚀 Getting Started
Install dependencies:

bash

yarn install
Start the development server:

bash

yarn start
🧰 Tech Stack
React

Yarn (package manager)

Axios – HTTP client

Lodash – Utility functions

FontAwesome – Icons

Normalize.css – CSS normalization
